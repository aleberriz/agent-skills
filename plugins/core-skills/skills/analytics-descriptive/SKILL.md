# Descriptive Analytics

*This skill is planned but not yet implemented.*

Conventions for the core data analyst workflow: taking a business question from intake through analysis to a stakeholder-facing deliverable.

When complete, this skill will cover:

- Analysis output structure: question, method, finding, implication
- Stakeholder communication patterns: executive summary, technical appendix, recommendation framing
- Reliability and confidence framing: how to communicate uncertainty without hedging into uselessness
- Notebook-to-deliverable workflow: exploration, validation, artifact
- Dashboard design heuristics: when to build a dashboard vs. when a one-off analysis is the right artifact
