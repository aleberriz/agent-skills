# Experimentation

*This skill is planned but not yet implemented.*

Conventions for rigorous experimental design in product and growth analytics. This is the causal inference domain where randomization is feasible.

When complete, this skill will cover:

- Power analysis and sample size calculation
- Randomization strategies: user-level, session-level, cluster
- Sequential testing and early stopping rules
- Metric selection: primary, secondary, guardrail
- Result interpretation: statistical significance vs. practical significance
- Common pitfalls: peeking, multiple comparisons, novelty effects, survivorship bias
